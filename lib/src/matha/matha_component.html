对于二次函数来说，其表示方法一般有三种，即数表法、图像法、关系式法，这三种表示方法彼此之间不但有联系，而且在某些情况下还可以互相转化.

关系式法：

用关系式表示二次函数时，一般表示为 y=ax^​2+bx+c (a,b,c 为常数，a≠0) 的形式，这个式子也叫作二次函数的解析式或表达式．

数表法：

x … -3 -2 -1 0 1 2 3 …
y=x^2-2x … 15 8 3 0 -1 0 3 …

定义剖析：

  • 二次函数 y=ax^​2+bx+c (a≠0) 中 x、y 是 变量，a、b、c 是常量
  • 自变量 的最高次数必须是 2，也就是说在 y=ax^​2+bx+c (a≠0) 中 ，而 b，c 可以为 0；
  • 含自变量的 代数式 是 整式，而不是分式或根式
  • 任何一个二次 函数 的关系式，都可以化成 y=ax^​2 ​​ +bx+c (a，b，c 为常数，a≠0) 的形式，因此 y=ax^​2 ​​ +bx+c (a，b，c 为常数，a≠0) 叫做二次函数的一般形式

函数值：

当给定 自变量 x 的一个值之后，就有唯一的 y 与之对应，这是相应的 y 值就是 函数 值．

例如：对于 二次函数 y=x^​2+2x−3 来说，当 x=2 时，函数值 y=2^​2+2⋅2−3=5，而当 x=-2 时，函数值 y=(−2)^2+2⋅(−2)−3=−3

注意

对于二次函数中的自变量取值范围内的每一个自变量的值，都有唯一的函数值与之对应，反之不一定

  • 如函数 y=x^​2−2x+1, 当 x=0 时，y=1
  • 反过来，若 y=1，可得 y=x^2−2x+1=1，这个一元二次方程有两个根 x​1=0，x2=2
